.. _d:resourcetype:

ResourceType (O)
----------------
The type(s) of the resource (occurrence: 0-n). Free text values allowed.

**Examples**

Dataset; text; survey data

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <resourceTypes>
      <resourceType>Dataset</resourceType>
      <resourceType>AstroImage</resourceType>
      <resourceType>DataCollection</resourceType>
   </resourceTypes>
