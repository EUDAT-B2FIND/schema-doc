Size (O)
--------
Size information about the resource (occurrence: 0-n). 

**Allowed values, examples, other constraints**

Free text. Examples: 123 MB, 3 pages, 5 cm.

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <sizes>
      <size>400 MB</size>
      <size>102 files</size>
   </sizes>
