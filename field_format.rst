Format (O)
----------------------
Technical format of the resource. Use file extension or MIME type where possible (occurrence: 0-n).

**Examples**

PDF, XML, MPG or application/pdf, text/xml, video/mpeg

Example
~~~~~~~
.. code-block:: xml
   :linenos:

  <formats>
    <format>
     PDF
    </format>
  <formats>
